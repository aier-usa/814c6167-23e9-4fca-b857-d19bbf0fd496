We provide an easy solution to the hard problem of password fatigue syndrome. Instead of stored encrypted password or plain password, our system stores password hints so that computer users have peace of mind.

Access points are public access entry point through which a computer user can make a password entry "public".

Main page when you open the site

This is the register page

After you have registered, you can log into the system

After you have logged in, you can click on Passwords button to see all your password entries

For one password entry, you can assign it to an access point for public view. BE CAREFUL. Once you have assigned, everyone in the world can view it.

Do you want to see all your access points?

For each access point, you can view it through ihatepassword.com/showme/XYZ. XYZ is the access point that you have created before.
